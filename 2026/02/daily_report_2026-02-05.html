2. 手机端：操作流程非常隐蔽。在发布界面，只能选择已有的合集，无法直接新建。新建合集的操作被隐藏在“我” -> 左上角三道杠 -> “创作中心” -> “更多服务” -> “合集管理”深处。

  • 首次创建合集：必须在手机端通过“创作中心”的路径预先创建好。
  • 日常添加笔记：发布笔记时，无论是在PC端还是手机端，都可以在发布页面的“添加组件”中找到并选择已创建的合集。

提问：科普类短视频如何优化，才能避免“干货劝退”？

爬楼关键词

高二壮不喝冰露开始痛风：新视频风格～一股营销号的味儿

解答
一、问题背景
高二壮老师分享了自己新制作的母婴科普短视频，并征求群友意见。视频采用了信息卡片滚动的形式，旨在传递高密度干货，但也引发了关于用户体验的讨论。

二、核心问题分析
群内专家从多个维度指出了视频的潜在优化点：
1. 视觉疲劳：橙子在写戏🍊老师指出，视频的黄色背景与黄色字幕颜色过于接近，即使用了描边，长时间观看也会让眼睛感到疲劳。
2. 听觉体验：束缚玉蝉老师认为背景音乐（BGM）声音太小，且缺乏特色。关于科普视频是否需要BGM，高二壮老师表示困惑，他担心音乐会干扰信息传递，但没有音乐又显得太干。
3. 信息过载与焦虑感：黄云老师提出了一个关键洞察——视频高密度的信息呈现方式，可能会让目标用户（宝妈）感到焦虑，并产生“需要来回拉进度条才能看清”的负面情绪，这不利于完播。
4. 结尾乏力：不渡老师评价视频结尾部分，感觉大脑已经放空，且缺少一个明确的“关注引导”，让用户的观看热情未能转化为关注行为。

三、解决方案与建议
针对以上问题，群友们给出了具体建议：
  • 优化视觉：调整字幕颜色，确保与背景有足够对比度，提升可读性。
  • 善用BGM：科普视频的BGM应选择“不抢镜”的类型，作为氛围铺垫，避免人声干涩，音量需适中。
  • 简化信息：黄云老师建议，短视频的核心是“注意力”，应简化信息呈现，避免做得像“博士论文”，让用户能轻松看完，而不是感到压力。
  • 强化结尾：在视频末尾设计明确的Call to Action（行动号召），直接告诉观众关注你能获得什么价值。
高二壮老师总结，新视频将尝试调整情绪、呈现形式，并借鉴毕导等优秀博主的风格，以更轻松、更有沉浸感的方式传递知识。
